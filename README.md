# GREEN-API WhatsApp Автотесты

Автотесты для проверки методов отправки и получения сообщений WhatsApp через GREEN-API.

## Требования

- Node.js 16+
- npm или yarn
- Аккаунт GREEN-API (https://green-api.com/)

## Установка

```bash
npm install
```

## Конфигурация

1. Зарегистрируйтесь на https://green-api.com/ и создайте инстанс
2. Получите `idInstance` и `apiTokenInstance` в личном кабинете
3. Отредактируйте файл `.env` и укажите ваши данные:

```env
ID_INSTANCE=ваш_id_инстанса
API_TOKEN_INSTANCE=ваш_api_токен
TEST_PHONE_NUMBER=79001234567
```

Где:
- `ID_INSTANCE` - ID вашего инстанса (например: 1101000001)
- `API_TOKEN_INSTANCE` - API токен инстанса
- `TEST_PHONE_NUMBER` - номер телефона для тестов в формате 79001234567

## Запуск тестов

```bash
npm test
```

```
src/

```

## Покрытие тестами

- ✅ Проверка статус кода 200 для успешных запросов
- ✅ Проверка статус кода 400 для некорректных запросов
- ✅ Проверка обязательных полей в запросах
- ✅ Проверка авторизации инстанса перед операциями
- ✅ Проверка обязательных полей в ответах

## Методы API

- `getStateInstance` - проверка состояния инстанса
- `sendMessage` - отправка текстового сообщения
- `getChatHistory` - получение истории чата

## Примечания

- Перед запуском тестов убедитесь, что ваш инстанс авторизован (отсканирован QR-код)
- Тестовый номер телефона должен быть в формате без + и пробелов
- Для работы getChatHistory нужна история сообщений с указанным номером
